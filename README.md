This is the "Brain Games" project, which contains a set of five console games built on the principle of popular mobile applications for pumping up the brain.

## Start ##
``` make 
```

## Setup ##
``` make build ```

## Run ##
``` make run ```

## Run tests ##
``` make test ```

## Run checkstyle ##
``` make lint ```

## Check update dependencies and plugins ##
``` make update ```
